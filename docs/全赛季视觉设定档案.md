# 《岚城联：一季之内》全赛季视觉设定档案

本档案补充第二至第十集的视觉事实边界。剧情正文、对话、选择、条件和后果始终以 `story-data.json` 为唯一来源；本档案中的外貌、材质、灯光与构图只属于视觉层，不能反向成为新剧情。

## 全局连续性

- 沿用序章的高品质日系动漫视觉小说风格：成年人真实比例、清晰线稿、赛璐璐与柔和厚涂结合、电影化光影。
- 岚城联是一家中等规模沿海城市职业俱乐部，专业、长期使用、并不奢华；俱乐部空间统一使用深绿、灰白、旧金色和实用混凝土材质。
- 背景不承载必须阅读的文字，不出现真实俱乐部、品牌、错误队徽、水印或乱码；重要信息只由原剧情文字表达。
- 同一人物跨章节保持脸型、发型、年龄、服装体系和体态一致。动态路线中的“主教练”和“队长”由既有决定决定，不新增剧情判断。
- 视觉素材缺失或配置失败时必须显示完整文字版，不能阻断推进或改变选择。
