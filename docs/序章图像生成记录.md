# 《你受谁委任》图像生成记录

本批素材使用内置 GPT Image 2 生成，只服务于第一集。所有人物与场景事实以 `story-data.json` 和《序章视觉设定档案》为准；下面记录的是最终采用的提示词集合，方便以后独立重做画面而不改剧情。

## 统一画风提示词

高品质日系动漫视觉小说插画，清晰而克制的线稿，赛璐璐与柔和厚涂结合，电影化光影，当代中国职业足球职场剧，成年人真实年龄与职业比例，深绿、灰白和少量旧金色的统一配色。专业但不奢华，不要校园感、幼态脸、Q版、真实俱乐部队徽、品牌、水印或可读乱码。

## 人物母版提示词

每名人物生成一张三联表情设定图：同一脸型、发型、服装、比例与视角，腰部以上，正面略侧身，纯洋红背景，无文字、无道具遮挡、无阴影落在背景。生成后去除洋红并切分为透明 WebP 立绘。
