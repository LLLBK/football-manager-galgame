# 《岚城联：一季之内》核心体验重构蓝图 v1

状态：方案稿，尚未进入程序修改
范围：人物系统、核心游玩循环、足球新手解释层、金钱压力、比赛因果反馈、前三集逐场重构
目标：把游戏从“阅读管理材料后做选择”改造成“面对有情感、有目的的人，在真实约束中作出会留下痕迹的决定”

---

## 一、已经确定的四条设计原则

1. 每个主要人物都是**诚实但有立场的说服者**。他们不提供假信息，但会按照自己的利益、责任、恐惧和价值排序来选择事实、组织事实、解释事实。
2. 玩家必须能够感到清晰的因果链，但不直接看到隐藏加减分。游戏告诉玩家“什么正在起作用”，不告诉玩家“防守 +12”。
3. 俱乐部允许真正失败。现金不足、承诺破裂、关系恶化和竞技失利必须关闭机会、迫使妥协，严重时可以造成欠薪、被迫出售、融资丧失控制权或玩家离任。
4. 先重做前三集，验证人物、金钱和比赛反馈是否成立，再扩展到全季。现有十集主题、主要人物、美术资源和关键选择尽量保留，不先做无必要的全面推翻。

## 二、玩家体验的北极星

完成前三集以后，玩家应该能自然说出：

- “周绍庭想要的不是单纯赢球，他想让俱乐部变成可以升值、可以出售的资产。”
- “方雯不是反对花钱，她害怕俱乐部再次在工资日之前没有选择。”
- “韩立锋不是保守的知识点，他害怕管理层用新口号夺走他的权威，输球以后再让他背锅。”
- “许青禾并不是逢商业必反，她不能替八千多名会员私下卖掉他们投票保留的东西。”
- “我买来的中卫在比赛里确实救过一次球，但为了注册他清走的替补，也让球队后半场没人可换。”
- “比分有偶然性，可这支球队为什么以这种方式赢、平或输，是我一路经营出来的。”

如果玩家只能记住“选项 A 加防守、选项 B 加青训”，重构就没有成功。

## 三、人物不再是信息接口：诚实的利益相关者系统

### 3.1 每个主要人物必须具备的字段

人物数据不再只有姓名、职位、简介和一项信任值。每个人至少要有以下内容：

| 字段 | 作用 |
|---|---|
| `publicDuty` | 他公开承担的职责，决定哪些诉求具有正当性 |
| `privateWant` | 他本人真正想得到的结果 |
| `fear` | 他最怕重演或最怕被证明的事情 |
| `stakes` | 本季结果会让他具体失去什么 |
| `preferredOutcomes` | 他会努力说服玩家选择的路线 |
| `redLines` | 触碰后，他会从劝说转向抵抗、退出或公开施压 |
| `truthLens` | 他优先注意哪些真实事实，又容易忽略什么 |
| `persuasionStyle` | 他如何说服人：数字、权威、故事、交换、羞耻、程序等 |
| `speechRules` | 句长、词汇、语速、是否使用比喻、是否直接说“我” |
| `physicalHabits` | 紧张、愤怒、期待或撒不开口时的动作 |
| `relationshipBehaviors` | 关系变化后，他的行为怎样变化，而不是只换一句状态文案 |
| `seasonArc` | 他在全季中可能怎样改变自己的看法与策略 |

### 3.2 不再使用单一“好感度”解释关系

人物关系使用少量离散行为状态和记忆标记：

#### 行为状态

- `open`：愿意在坏消息公开前先告诉玩家。
- `guarded`：只回答被问到的内容，不主动补充。
- `oppositional`：仍然说真话，但会组织他人共同施压或公开表达异议。
- `withdrawn`：不再相信私下沟通能改变结果，改走正式程序、媒体、董事会或经纪人。

#### 过程记忆

- `consulted`：在与其职责直接相关的决定前被认真询问。
- `ignored`：在与其职责直接相关的决定中被绕过。
- `promiseKept`：玩家曾兑现对他的明确承诺。
- `promiseBroken`：玩家违背或拖延对他的明确承诺。
- `burdenShared`：坏结果出现时，玩家没有把责任甩给他。
- `usedAsShield`：玩家利用他的意见替自己挡责。

同一个选择造成的反应不能只看“是否顺着他”，还要看过程。例如玩家可以拒绝方雯的建议，但如果让她完整陈述风险、公开承认这是自己的决定，她可能失望却仍愿意继续直说。相反，表面上采用她的方案，却把责任说成“财务部不让买人”，会伤害关系。

### 3.3 人物提供的事实必须全部真实，但不必完整中立

规则如下：

1. 人物不能伪造数字、合同、伤情或投票结果。
2. 人物可以只强调对自己主张有利的真实部分。
3. 人物必须承认直接追问后暴露的真实代价。
4. 玩家若同时听取立场冲突的两人，应能拼出更完整的因果链。
5. 人物的说服不能永远成功；被拒绝后，他会调整策略，而不是在下一集恢复成中立信息员。

例如韩立锋会真实地说明缺中卫如何破坏训练计划，却不会主动先谈清理两名替补对阵容深度的伤害。方雯会真实地说明冬天的现金缺口，却可能低估一次关键引援对教练权威和成绩压力的价值。两个人都没有骗人。

## 四、核心人物圣经

### 4.1 周绍庭｜俱乐部主席

**公开职责**
确保俱乐部持续经营、董事会投入得到回报，并对最终成绩负责。

**真正想要**
在两年内把岚城联变成一项更值钱、更容易引入投资者或出售的资产。他希望球队有一个市场听得懂的上升故事：前六、年轻资产、球场商业化或名帅计划都可以，唯独不能长期含混。

**害怕什么**
害怕五年前救俱乐部的投入变成无法退出的情感债；也害怕别人把他定义成“只会花地产商的钱、不懂足球的老板”。

**本季利害**
若现金和排名同时恶化，他可能失去董事会支持，出售意向也会贬值。若俱乐部升值，他能证明自己的管理方式有效。

**偏好**
公开目标、明确负责人、可传播的转折点、能快速提升资产价值的决定。

**红线**
玩家要求权力，却拒绝承担公开结果；或者用协商拖延所有决定。

**真实视角**
最先看到资产价值、声誉、投资者预期和董事会耐心；容易把人的长期关系当成可以通过新成绩覆盖的旧成本。

**说服方式**
把模糊问题改写成期限和指标；常用“如果……那么谁负责”迫使对方表态。他不提高音量，越不耐烦越简短。

**语言规则**
短句、结论先行、少形容词；喜欢使用“给我一个日期”“谁签字”“最坏情况是什么”。避免哲理式金句。

**动作习惯**
用钢笔压住不想让别人先看的文件；听到没有期限的计划时会拧笔帽；满意时不是微笑，而是把文件推回给对方签字。

**关系变化**

- `open`：提前透露董事会内部交易、潜在投资者和自己的真实容忍线。
- `guarded`：只通过正式会议下达目标，不再提前给玩家调整空间。
- `oppositional`：绕过玩家联系教练或经纪人，为替代方案做准备。
- `withdrawn`：启动出售、解雇或权力收回程序。

**前三集想说服玩家做什么**

- 第一集：签董事会目标书，把“前六”变成全俱乐部公开故事。
- 第二集：优先即战力，因为公开目标需要一个能被看见的投入证明。
- 第三集：他不真正关心阵型，但偏好能快速制造变化且便于宣传的主动打法；若玩家能证明韩立锋的方案同样有清晰指标，他也会接受。

### 4.2 方雯｜财务负责人

**公开职责**
保证工资、分期、税费和比赛运营按期支付，让董事会知道每个承诺何时会变成现金支出。

**真正想要**
建立一条任何主席、教练或经理都不能越过的现金纪律，让俱乐部不再依赖临时救火。

**害怕什么**
她经历过一次延迟发薪：行政员工先被要求“共克时艰”，球员经纪人却在媒体得到消息。她害怕同样的事情再次发生，也害怕自己成为那个太迟说“不”的人。

**本季利害**
若现金链断裂，最终打电话向员工解释的人是她；若她总是阻止投入，她也可能被视为妨碍竞技的保守官僚。

**偏好**
现金安全垫、分期透明、风险附带触发条件、在支出前写明补救方案。

**红线**
用未来不确定收入支付今天的固定义务；或者把玩家自己的选择对外说成“财务部不批准”。

**真实视角**
最先看到到期日、现金错配和隐藏义务；容易低估错过竞技窗口、伤害教练权威和球迷期待造成的非财务损失。

**说服方式**
不用“贵”“便宜”，只说具体日期和届时缺口。她会让对方亲手在付款日历上划掉一笔钱。

**语言规则**
句子克制，金额后面必须跟日期或来源；不说抽象的“财务健康”，说“十月十五日工资日还差多少”。

**动作习惯**
开会先关空调，因为机器声会妨碍她听清数字；压力大时把文件按到边缘完全对齐；真正生气时不再用便签，改发正式邮件并抄送董事会。

**关系变化**

- `open`：在风险尚可挽救时提前找玩家，也会提供不体面的融资备选方案。
- `guarded`：只提供合规数字，不再主动提出补救路径。
- `oppositional`：要求决定写入会议纪要并让主席共同签字。
- `withdrawn`：拒绝为缺乏资金来源的合同签署财务确认。

**前三集想说服玩家做什么**

- 第一集：不要作出没有付款路径的公开承诺；若必须承诺，写清复核日期。
- 第二集：至少保留能覆盖秋季工资和一月分期的现金，不反对花钱，但要求每项支出带补救方案。
- 第三集：支持与现有阵容、未来工资能力相匹配的比赛模型，反对一句口号引出整套昂贵换血。

### 4.3 韩立锋｜一线队主教练

**公开职责**
训练和选择一线队，在比赛日对结果负责。

**真正想要**
获得一个完整赛季证明自己的方法不是保级应急方案，而是一套可以长期赢球的足球。

**害怕什么**
害怕管理层把新打法、青年球员和数据部门塞给他，成绩不好时却只留下他的名字。他把外部建议理解成未来甩锅的准备。

**本季利害**
合同只剩一年。若开局失利，他可能成为最先被牺牲的人；若球队稳定，他能从救火教练变成真正的建队者。

**偏好**
主教练边界清晰、成熟球员、控制风险、球队知道谁做最后决定。

**红线**
玩家在媒体面前替他决定首发、公开否定训练方案，或承诺保护期后因一场失利立即转向。

**真实视角**
最先看到阵容缺口、训练负荷和球员是否执行；容易把医疗、青训和财务对长期风险的提醒理解成对自己权威的侵入。

**说服方式**
用具体比赛连锁反应说服人，不讲宏大理念。他会把“不给我中卫”翻译成“梁一川回撤、程野少上、压迫取消”。

**语言规则**
口语短促，多用球场动作词；不说“组织稳定性”，说“谁丢球以后往回跑”；被冒犯时不辩论理念，只问“最后谁负责”。

**动作习惯**
谈阵容时不断移动战术磁块；生气时把名单折起来；真正信任玩家时允许其站进训练场内圈，而不是口头感谢。

**关系变化**

- `open`：主动报告球员情绪和自己方案的弱点。
- `guarded`：只报告训练结果，不再邀请玩家进入更衣室或录像会。
- `oppositional`：通过记者、队长或主席迫使玩家公开表态。
- `withdrawn`：开始保存邮件、训练计划和伤病意见，为离任与自保留档。

**前三集想说服玩家做什么**

- 第一集：确认足球部门边界，不在媒体前越级。
- 第二集：签下中卫，兑现竞争目标。
- 第三集：完整支持控制型足球，并承诺输球后仍由他定义问题。

### 4.4 许青禾｜球迷信托理事

**公开职责**
执行8437名缴费会员通过的决议，而不是代表抽象的“球迷情绪”。

**真正想要**
让球迷从被通知、被拍照的背景，变成重大资产决定中有正式权力的一方。

**害怕什么**
她父亲曾相信俱乐部“临时”更改的承诺，最后一项也没有恢复。她害怕自己因为能进入会议室，反而被俱乐部用来证明“已经听取球迷意见”。

**本季利害**
如果她私下接受越过会员投票的交易，会失去会员授权；如果她拒绝所有商业方案，也可能被指责让看台继续漏雨。

**偏好**
正式表决权、可撤销条款、低价票、东看台名称和壁画得到保护。

**红线**
在投票前签署不可逆合同；引用她的出席来宣称球迷已经同意。

**真实视角**
最先看到决定对共同记忆、普通球迷和长期合法性的伤害；容易低估紧急现金和执行速度。

**说服方式**
带来具体的人、旧票、投票数字和会议决议。她区分“我个人愿不愿意”和“我有没有权替会员答应”。

**语言规则**
不说泛泛的“传统”，说具体名称、票价、座位和投票；情绪越强烈，越强调程序和授权，避免抒情长句。

**动作习惯**
把旧季票夹在会员决议里；被敷衍时不争吵，只拿出手机记录原话；真正愤怒时会把座位让给另一名会员代表，拒绝继续成为唯一可被俱乐部控制的“球迷声音”。

**关系变化**

- `open`：提前告知球迷内部的真实分歧和可接受妥协。
- `guarded`：只转达正式决议，不再替玩家向会员解释。
- `oppositional`：组织联合声明、公投或现场行动。
- `withdrawn`：退出非正式协商，要求所有沟通公开记录。

**前三集想说服玩家做什么**

- 第一集：采用共同章程，承认球迷信托在重大资产问题上的正式地位。
- 第二集：她不是预算方案主角，但会质问公开目标是否要靠透支共同资产完成。
- 第三集：不指定踢法；她希望球队至少形成普通球迷能够认出的、不会两场一变的身份。

### 4.5 梁一川｜队长 / 中场

**公开职责**
连接教练组与球员，在训练和比赛中维持秩序。

**真正想要**
体面地完成从绝对核心到老队长的过渡，同时确认俱乐部不会把“忠诚”当成少付钱、少说真话的理由。

**害怕什么**
害怕自己成为纪念品：海报上永远首发，现实里却没人敢当面谈年龄、合同和位置。

**本季利害**
合同只剩一年。他可能离开家乡球队，也可能留下却逐渐失去位置和尊严。

**偏好**
一致标准、当面沟通、教练与管理层共同承担决定、年轻人获得真正机会。

**红线**
用“终身忠诚”掩盖合同问题；在球员最后才知道涉及自己的安排。

**真实视角**
最先看到更衣室中的沉默、派别和自我保护；容易为了保护队友而延后向管理层报告冲突。

**说服方式**
不替球员喊口号，而是描述一个决定在训练场会变成什么动作。他常把争论改写成“什么错误会被允许”。

**语言规则**
少用管理词，使用球员能复述的话；不轻易直接要求利益，常先谈队内后果。

**动作习惯**
最后离开录像室；谈自己合同时摸袖标边缘；真正失望时回复“收到”，不再补充意见。

**前三集想说服玩家做什么**

- 第一集：不要制造三个互相冲突的权威中心。
- 第二集：他理解补中卫的竞技价值，也会注意两个被清走的柜子由谁承担。
- 第三集：选择一种输球后仍然算数的评价标准。

### 4.6 罗志衡｜青训主管

**公开职责**
招募、培养并保护青年球员进入职业队的路径。

**真正想要**
证明青训不是廉价球员仓库，而是一套需要俱乐部主动承担失败和生活成本的制度。

**害怕什么**
害怕俱乐部只展示程野这一名成功者，却让家庭资源较少的孩子在交通、住宿和学业中悄悄消失。

**本季利害**
宿舍和梯队机会若继续推迟，他会失去下一批球员，也可能失去对俱乐部承诺的信心。

**偏好**
宿舍、教育、明确晋升标准、各级球队共用可迁移的比赛原则。

**红线**
把年轻球员当宣传素材，却不给训练反馈和真实出场路径。

**真实视角**
最先看到家庭背景、成长时间和人才流失；容易低估一线队短期成绩对全俱乐部生存的影响。

**说服方式**
带来具体孩子和具体日程，不用“青训未来”空话。他会告诉玩家谁每天几点起床、坐几小时车。

**语言与动作**
说话耐心，但一涉及孩子被当资产就会纠正称呼；习惯随身带训练记录本，紧张时翻到具体日期而不是争辩理念。

### 4.7 程野｜18岁青训前锋

**公开职责**
几乎没有。他在组织里权力最小，却最容易被所有路线拿来证明自己正确。

**真正想要**
知道自己怎样才能上场，而不是永远被称为“未来”。

**害怕什么**
害怕一次失误就证明自己不配，也害怕俱乐部只是用他的照片证明重视青训。

**本季利害**
经纪人已经接到电话。留队可能没有比赛，离开可能失去熟悉的支持系统。

**偏好**
明确反馈、可预期的训练与出场路径、允许犯错。

**红线**
把他写进公开承诺，却从不告诉他距离一线队还差什么。

**真实视角**
最先感到教练的细小态度和队友是否还愿意传第二次球；不了解财务和董事会全貌。

**说服方式**
初期不擅长直接说服。他会通过训练、短讯和不敢发出的视频让玩家看到代价。关系变好后，才会直接问“我到底排第几”。

**语言与动作**
句子短，有时答非所问；紧张时拉训练服下摆、确认号码贴；高兴时也不说宏大感言，只问下一次训练几点。

### 4.8 孟书宁｜队医

**公开职责**
保护球员健康，提供独立、准确的可出场判断。

**真正想要**
让球员相信如实报告疼痛不会损失位置，让医疗意见不会因比赛重要而改写。

**害怕什么**
害怕某次自己妥协的“可控风险”最终改变一个人的职业生涯；也害怕被教练和球员视为不懂竞争的人。

**本季利害**
若医疗意见持续被越过，她会失去球员的真实报告；若她拥有权力，也必须承担球队短期减员的公开压力。

**偏好**
独立报告权、伤病奖金保护、提前干预、恢复团队建设。

**红线**
要求修改伤情等级；让球员在位置和奖金威胁下签署所谓自愿文件。

**真实视角**
最先看到身体风险和隐瞒伤情；容易低估一场比赛对合同、教练职位和俱乐部收入的巨大影响。

**说服方式与语言**
说坏消息很慢，逐项确认对方听懂；不用“绝对安全”，只说概率、时间和不可逆后果。紧张时会把病历合上，确认房间里谁有权听。

### 4.9 高竞｜待业名帅

**公开身份**
尚未进入俱乐部，没有正式职责。

**真正想要**
接手一支愿意围绕自己重建、能够证明上一段失败不是其方法问题的球队。

**害怕什么**
害怕只被当作给现任教练施压的名字，或者接手后得不到承诺的阵容资源。

**偏好与红线**
偏好高强度、快速换血、教练团队完整进入；拒绝只在海报上采用其“主动足球”，实际不给训练时间和球员。

**说服方式**
前三集不应像中立战术方案一样出现在案头。他应通过中间人、短讯或一段明确带条件的通话存在：“要我承担高位风险，就把六周训练和两名助手写进合同。”

## 五、人物偏好不是隐藏答案：前三集立场矩阵

| 人物 | 第一集偏好 | 第二集偏好 | 第三集偏好 | 可能接受的妥协 |
|---|---|---|---|---|
| 周绍庭 | 董事会目标书 | 即战力 | 能快速产生清晰故事的路线 | 只要期限、负责人和传播逻辑明确，可接受其他路线 |

## 六、新的核心游玩循环

### 6.1 旧循环

```text
阅读两三段背景说明
→ 从三个人中选择两人询问
→ 得到三条中立化事实
→ 阅读三张政策卡
→ 三选一
→ 看两段预写余波
→ 下一集
```

这个结构每集完全相同，人物被压缩成资料来源，玩家也很容易进入“找正确答案”的答题状态。

### 6.2 新循环

```text
一个已经在发生的冲突
→ 有立场的人主动争取玩家
→ 玩家选择回应方式，而不只是选择听谁
→ 通过追问、反问或交换条件暴露真实代价
→ 玩家说出/签下具体决定
→ 立即看到人、钱、承诺和机会发生变化
→ 后续场景与比赛把这些变化重新带回来
```

### 6.3 每集不再强制使用同一种互动模板

- 第一集是“最后五分钟你去哪里”：董事会议室、楼梯口或训练场。空间选择同时意味着关系选择。
- 第二集是“拆预算包”：所有主张都已摆在桌上，玩家可以质疑一项假设，并要求一方作出让步。
- 第三集是“看录像”：玩家点击具体比赛片段，人物对同一画面给出不同解释，再决定允许球队犯哪一种错误。

每集仍可保持可测试的有限分支，但玩家表面体验不再是同一种三选一问卷。

### 6.4 决定前必须让玩家知道四件事

不显示隐藏数值，只显示：

1. **谁在要求这件事。**
2. **这件事立刻改变什么。**
3. **你在押注哪个未来条件。**
4. **谁会承担最坏情况。**

示例：

> **签下主力中卫**
> 韩立锋要求；周绍庭支持。
> 今天支付1400万，并清理两份边缘球员合同。
> 你押注新中卫带来的防守稳定，足以抵消替补深度与冬季现金的风险。
> 如果密集赛程出现伤病，剩余球员和冬窗预算承担后果。

### 6.5 决定后使用“后果回执”而不是属性提示

每次决定后固定显示四类变化中的相关项：

- **已经发生**：签字、付款、人员离开、角色改变。
- **已经答应**：玩家原话、对象、到期日。
- **已经埋下**：未来付款、阵容缺口、程序争议、伤病风险。
- **谁改变了做法**：人物下一步行动，而不是抽象好感变化。

## 七、足球新手解释层

### 7.1 原则

1. 默认开启“第一次经营足球俱乐部”解释层，设置中可关闭。
2. 术语第一次出现时可点击或轻触，不强迫离开剧情阅读长词典。
3. 每条解释包含“它是什么”和“为什么此刻重要”。
4. 正文优先使用白话，专业词只在能增加真实感时出现。
5. 同一个术语不反复弹出；人物对术语的解释也要符合其说话方式。

### 7.2 前三集最小术语表

| 术语 | 白话解释 | 此刻为什么重要 |
|---|---|---|
| 更衣室 | 不只指换衣服的房间，也指球员内部关系、情绪和权威秩序 | 球员若不再彼此信任，可能不补位、不报伤、也不替教练承担风险 |
| 首发 | 一场比赛开始时先上场的十一名球员 | 首发位置关系到比赛时间、奖金、地位和球员是否相信教练 |
| 轮换 | 在不同比赛中更换部分首发，让更多球员上场并控制疲劳 | 阵容太薄时，球队无法轮换，后半程更容易疲劳和受伤 |
| 中卫 | 站在防线中央、主要阻止对手靠近球门的球员 | 岚城联缺少中卫，会迫使队长回撤并改变整套踢法 |
| 中低位 | 球队整体站得更靠近自己球门，压缩防守空间 | 更稳定省体能，但落后时可能难以主动制造机会 |
| 高位压迫 | 在靠近对方球门的位置集体追抢，争取快速夺回球权 | 能制造进攻，却消耗体能，也会在自己身后留下空间 |
| 定位球 | 角球、任意球等比赛暂停后重新开始的进攻机会 | 进攻创造力有限的球队常靠它决定比分 |
| 注册 | 联赛正式允许一名球员代表俱乐部上场 | 有钱签合同不代表联赛一定允许他进入比赛名单 |
| 球队成本限额 | 联赛限制俱乐部可用于球员工资、签字费和转会摊销的额度 | 岚城联现金够，但注册额度不足，必须先清理旧成本 |
| 转会分期 | 买球员时没有一次付清、未来仍要支付的钱 | 今天买人会占用冬天和下个赛季的现金 |
| 工资承诺 | 已签合同未来必须支付的工资，不是今天一次付掉的钱 | 长约会让未来每个工资日都更紧张 |
| 受限资金 | 账上存在但只能用于特定用途的钱 | 600万青训专款不能拿去支付引援或普通工资 |

## 八、金钱系统：从数字改成会伤到人的时间表

### 8.1 玩家真正需要看到的不是余额，而是生存时间

财务面板改为：

```text
今天可动用：3600万

未来90天
秋季工资与签字费：−1200万（已确定）
一月旧转会分期：−900万（已确定）
赛季奖金准备金：−400万（已确定）

若不再支出：已知付款后剩1100万
若签下中卫：一月付款后只剩100万；赛季奖金到期前将出现300万缺口
```

金额必须同时带日期、来源和具体后果，不能只写“现金 −1400”。

### 8.2 现金状态

- **有选择空间**：覆盖未来已知付款后仍保留1000万以上。
- **脆弱**：覆盖未来已知付款后只剩400—999万；一次伤病或收入延迟就会触发危机。
- **临界**：覆盖未来已知付款后只剩0—399万；新支出必须同时附融资或削减方案。
- **资不抵期**：预测余额低于0；不能把决定简单执行为负数，必须先进入危机事件。

玩家不需要看到这四档背后的“风险分”，但要看到具体哪一天、哪一笔钱会出问题。

### 8.3 硬约束

1. 立即现金不足的选项不可执行，除非选项本身包含确定融资。
2. 到期付款不足时，不允许余额静默变负，必须触发危机选择。
3. 受限资金不可自动挪用；挪用必须是明确违规或重新审批剧情。
4. 新增长期工资、分期或借款必须进入付款时间表。
5. “预计赞助”“可能卖人”等不确定收入不能自动用于覆盖固定付款。

### 8.4 现金危机的真实后果顺序

危机不应立即等于游戏结束，而应迫使玩家接受越来越差的选择：

1. 冻结非必要采购和青训/医疗项目。
2. 延迟供应商付款，服务质量下降。
3. 请求股东过桥借款，附带董事会权力或出售条件。
4. 被迫出售球员，买方因知道你急售而压价。
5. 延迟工资或奖金，球员、员工和经纪人改变行为。
6. 注册限制、转会禁令、董事会撤换玩家。

### 8.5 钱必须通过具体的人和物被感知

- 花掉450万不是“青训 +12”，而是三名孩子不再每天坐四小时车。
- 花掉1400万不是“防守 +12”，而是一个新中卫进入柜子、两个边缘球员清空柜子。
- 省下恢复团队的钱，不是“体能 −3”，而是孟书宁只能在比赛后临时借设备。
- 赚到赞助费，不是“现金 +950”，而是球员必须参加拍摄、看台改名、企业票挤占低价票。

## 九、比赛系统：比分有偶然性，比赛故事有来源

### 9.1 不再把五项能力简单平均

每项能力承担不同职责：

| 因素 | 在比赛中的职责 |
|---|---|
| 进攻 | 产生多少次有威胁进攻、能否在落后时主动创造机会 |
| 防守 | 限制对手机会、处理禁区和身后空间 |
| 体能 | 后半场能力衰减、压迫持续时间和伤病概率 |
| 凝聚力 | 失误后的补位、连续丢球风险和比赛表现波动 |
| 士气 | 落后后的反应、关键时刻是否继续执行计划 |
| 教练权威 | 临场调整是否被理解和执行 |
| 阵容深度 | 是否能通过换人维持体能、处理伤病或改变比赛 |

阵容深度需要成为独立竞技状态，不能继续被隐含在其他数值里。

### 9.2 一场比赛的生成顺序

1. 读取本队比赛原则和对手特点，形成对位风险。
2. 生成4—6个关键比赛事件，而不是先抽出最终比分。
3. 每个事件由一个主要因素、一个次要因素和可能的随机因素共同决定。
4. 事件累积形成机会与进球，最终得到比分。
5. 每个事件保留来源：哪项状态、哪次决定、哪项承诺或哪次随机事件参与了它。

### 9.3 随机性必须有名字

随机事件可以是：

- 暴雨导致球速变化。
- 裁判尺度突然严格。
- 折射改变射门方向。
- 对方门将超常发挥。
- 年轻球员完成一次超出预期的动作。

赛后要明确标记“不可控”，避免玩家把所有结果都归因于自己的选择；但不可控事件不应吞掉整场比赛的解释。

### 9.4 赛后因果报告

每场至少返回：

- 两个旧决定带来的帮助。
- 一个旧决定带来的代价或风险。
- 一个不可控因素。
- 一句教练或球员基于自身立场的解释。

示例：

```text
第18分钟｜新中卫抢在对方前锋之前解围
来源：第二集签下主力中卫。

第71分钟｜右后卫抽筋，替补席没有同位置球员
来源：为清理注册空间外租两名边缘球员；阵容深度不足。

第84分钟｜湿滑草皮让门将第一次扑救脱手
来源：暴雨，属于不可控事件。

结果：1比1
韩立锋认为新中卫证明投入正确；方雯提醒一月前只剩100万安全余量。
两句话都是真的。
```

### 9.5 选择影响的是概率与比赛形状，不保证比分

买中卫不会保证不丢球；它提高特定防守事件成功率。主动压迫不会保证进球；它增加前场夺回球权的机会，同时增加体能消耗和身后暴露。玩家经营的是风险结构，不是答案按钮。

## 十、前三集逐场重构蓝图

## 第一集：你受谁委任

### 本集功能

- 让玩家在十分钟内感到三股互相冲突的授权：老板、球队、球迷。
- 让周绍庭、韩立锋、许青禾主动说服玩家，而不是等玩家点名询问。
- 第一次建立“过程也会被记住”：最后五分钟去见谁，会影响另两人的行为。
- 不讲完整经营规则，只让玩家理解自己的职位不是“教练”，而是决定资源、合同和组织边界的人。

### 场景1：董事会议室｜钢笔没有墨

**在发生什么**
周绍庭要求玩家在任命书旁同时签一份前六目标书。出售意向书的一角从文件下露出来。

**周绍庭想要什么**
让玩家在见到教练和球迷前先接受董事会定义的成功。

**情绪与动作**
他表现得礼貌、笃定；玩家问到出售意向时，他先给钢笔换墨，再回答。这表示他愿意谈，但谈话必须通向签字。

**玩家可采取的回应**

- “先把十二月以后会发生什么写清楚。”——逼他承认现金和排名的双重底线。
- “我可以负责，但不在今天报一个名次。”——争夺个人授权。
- “教练、队长和球迷代表还没进过这个房间。”——提出共同章程。

这些回应不是最终决定，但会改变周绍庭下一场的说服方式。

### 场景2：楼梯口｜许青禾没有被允许进门

**在发生什么**
许青禾不是偶遇。她曾申请旁听任命会议，被秘书以“商业保密”为由拒绝。她带着会员决议和父亲的旧季票等玩家。

**她想要什么**
不是一句“尊重球迷”，而是要求玩家承诺重大资产决定前给信托正式席位。

**她的说服**
她先说自己个人可以接受招商，再把会员决议翻到签字页：“但这个不能由我私下答应。”由此表现她有目的、有边界，也不是反商业机器人。

**足球新手提示**
首次解释“球迷信托”：由缴费会员选出的组织，代表会员对俱乐部公共事务表达意见；它不等于网上所有球迷的即时情绪。

### 场景3：训练场｜韩立锋不让你跨过白线

**在发生什么**
玩家到训练场时，韩立锋让助教把一条白色标志带重新拉直，没有邀请玩家进入内圈。

**他想要什么**
明确合同、引援和设施由玩家负责，首发、训练和更衣室由他负责；同时希望玩家给出是否公开支持他的答案。

**他的说服**
不讲“教练权威”。他指向正在训练的球员：“你今天替我说谁首发，明天他们训练时就先看你的脸。”

**足球新手提示**
首次解释“首发”和“更衣室”。解释直接贴在这句话旁边，不中断对话。

### 场景4：发布会前五分钟｜你最后去哪里

玩家只有时间完成一场最后谈话：

- 回董事会议室：周绍庭透露出售可能与十二月双重考核；韩立锋会记住玩家没有回训练场。
- 留在楼梯口：许青禾交出完整会员底线；周绍庭改为通过秘书催签。
- 去训练场内圈：韩立锋说出愿意替玩家挡住球员的交换条件；许青禾把未获回应写入会员简报。

没有被选择的人不会永久失去内容，但会进入 `ignored` 或 `guarded`，下一次不再用相同方式主动靠近。

### 最终决定

保留三条路线，但选项改写为玩家真正要说的话：

1. **“本赛季冲击前六，十二月由我接受复核。”**
   周绍庭得到想要的倒计时；韩立锋担心目标进入训练场；许青禾认为球迷仍在门外。
2. **“重大资产决定前，教练、队长和球迷信托都有正式席位。”**
   许青禾支持；方雯开始建立承诺表；周绍庭认为玩家给自己增加了程序约束。
3. **“给我一个完整赛季。足球决定由我负责，结果也只找我。”**
   韩立锋暂时接受边界；周绍庭只给时间不追加预算；许青禾无法确认玩家会不会成为另一个个人老板。

### 决定后的回执

除人物余波外，必须显示：

- 玩家原话与到期日。
- 谁认为自己被授权，谁认为自己被排除。
- 第二集会受影响的具体条件：引援报价、预算追加、协商程序或教练边界。

## 第二集：账上有钱，未来没有

### 本集功能

- 让玩家第一次真正理解余额、受限资金、注册额度和未来付款是不同的东西。
- 让钱通过三个具体的人生问题呈现：一线队缺中卫、青训孩子通勤、医疗团队没有独立资源。
- 决定后立即形成可见的90天现金预测和会在第三集比赛中出现的竞技后果。

### 场景1：财务办公室｜方雯让玩家亲手划掉2500万

**在发生什么**
方雯把3600万可用现金写在白板顶端，让玩家依次划掉秋季工资1200万、一月旧分期900万、赛季奖金准备金400万。只剩1100万。

**她想要什么**
先让玩家承认“3600万不是可自由花完的钱”，再讨论任何方案。

**交互**
玩家不是点击“问方雯”。玩家亲手点开三笔到期款，每点一笔，预测余额随日期变化。术语解释与付款对象同时出现。

### 场景2：训练场录像室｜韩立锋按暂停

**在发生什么**
韩立锋播放中卫候选人的一次解围，又切到岚城联热身赛中梁一川被迫回撤的片段。

**他想要什么**
把“不买中卫”从省钱选择改写成放弃整个夏训计划。他明确要求签人。

**他的盲点**
玩家追问清理哪两份合同时，他会停顿，承认替补深度受损，但说“替补不能替我们先活过前六轮”。这是真实、有立场的回答。

### 场景3：宿舍空房｜罗志衡带来三个家庭

**在发生什么**
不是读申请书。玩家看到三个还没铺床的房间和三张通勤表。程野母亲辞职接送的事实由程野自己不自然地说出来。

**罗志衡想要什么**
批准450万宿舍改造，并把青训从“有余钱再做”变成俱乐部必须承担的生活成本。

**他的盲点**
韩立锋问宿舍本季能贡献几分，罗志衡承认不能保证任何积分。他强调的是谁有资格留下来竞争。

### 场景4：诊疗室门口｜孟书宁没有申请一台机器

**在发生什么**
孟书宁把设备清单撕掉，改为要求两名康复师、独立伤情报告和否决高风险签约的权力。

**她想要什么**
玩家选择能力建设路线，或者至少在其他路线中写下冬季复核日期。

### 决策前互动：拆开一个方案的隐藏条件

三个方案都已经由支持者完整提出。玩家可以选择一个方案进行“压力测试”：

- 要求方雯说明最坏付款日。
- 要求韩立锋说出要清理的两份合同及竞技代价。
- 要求罗志衡说明宿舍两年内无法带来什么。
- 要求孟书宁说明独立否决权会怎样与教练冲突。

玩家不是为了找正确答案，而是逼说服者承认自己方案的真实代价。被压力测试的人如果仍被尊重，会提高对玩家程序公平的认可。

### 最终决定与真实财务后果

#### 路线A：先保住冬天

**决定**
支付450万改造宿舍；保留至少2000万现金安全垫；中卫只谈不增加长期承诺的租借。

**立即回执**

- 今日现金：3600 → 3150万。
- 已知付款后预计剩余：650万，进入“脆弱”状态。
- 三名青训球员获得住宿位置。
- 韩立锋转为 `guarded`；若玩家公开承担决定而不说“财务不批”，他不会进入 `oppositional`。
- 第三集球队仍缺速度型中卫，控制型打法更安全，主动高压的身后风险更高。

#### 路线B：把钱换成即战力

**决定**
支付1400万首款；清理两名边缘球员合同换取注册空间；签下主力中卫。

**立即回执**

- 今日现金：3600 → 2200万。
- 秋季工资与一月分期后预计只剩100万。
- 赛季奖金准备金到期前将出现300万缺口；玩家必须承认冬季卖人、融资或新增收入至少会发生一项。
- 新中卫进入首发竞争，防空与回追提高。
- 两个柜子清空，阵容深度下降；第三集比赛后半段更容易出现无人可换。
- 方雯若此前未被压力测试，直接转为 `oppositional` 并要求决定写入董事会纪要。

#### 路线C：投资不容易被看见的能力

**决定**
投入900万组建康复、球探与数据小组，增加明确的年度工资安排；不签明星。

**立即回执**

- 今日现金：3600 → 2700万。
- 已知付款后预计剩200万；新增团队工资还会继续消耗现金，进入“临界”状态。
- 医疗与数据部门可提前暴露风险。
- 韩立锋认为权力被切走一部分，但若玩家明确“数据可以否决风险、不能替你选首发”，他保持 `guarded` 而非 `oppositional`。
- 第三集比赛不会凭空增加进球，但能更早发现体能和阵容错配，并降低后半场失控概率。

### 第二集结尾

不直接播放抽象余波。结尾同时出现三个画面：现金日历、一个具体的人离开或进入房间、一张第三集比赛准备表。玩家知道钱已经改变下一场比赛的条件。

## 第三集：球队准备怎样赢

### 本集功能

- 用具体比赛片段解释中低位、压迫、阵容深度和定位球。
- 让韩立锋、梁一川、程野、罗志衡及数据人员对同一画面给出带立场的真实解释。
- 第一次完整展示“第一集授权 + 第二集资源 + 第三集比赛原则 → 场上事件 → 比分”。

### 场景1：封闭热身赛｜先让玩家看，不先讲理论

比赛0比0结束。玩家依次看到三个可点击片段：

1. 第22分钟，全队退到本方半场，连续封住三次向中路传球。
   解释“中低位”：站得更靠近自己球门，空间更小、更稳定。
2. 第57分钟，程野向对方后卫身后启动，持球队友没有传。
   解释“转换机会”：夺球后趁对方没站稳快速进攻。
3. 第81分钟，角球制造全场唯一一次禁区内射门。
   解释“定位球”：比赛暂停后重新开始的固定进攻机会。

先让玩家形成感受，再允许人物解释。

### 场景2：录像室争论｜同一场0比0有三种真相

**韩立锋**
指出三次封堵和对手零次单刀：“这场球到了八十分钟还属于我们。”他想证明控制方案有效。

**程野**
没有发表战术意见，只问分析师能不能把五次启动一起播放。他想证明自己不是“不参与比赛”，而是球没有来。

**梁一川**
不选边。他问韩立锋：“下一场先丢球，这套距离还算不算？”他想获得一致的错误标准。

**数据人员**
说明压迫不是“更勇敢”，而是需要六周训练、轮换和特定球员；同时说明现有控制型打法的进攻上限。数据人员不能给出推荐答案，只能比较条件。

### 场景3：高竞的条件，不再是一份中立摘要

若玩家打开高竞路线，中间人播放一段语音：

> “要防线前压，可以。六周训练、每场至少五个换人名额里有三个能真正用、身后丢球不能第二天就改口。缺一项，就不要把我的名字写在这套东西上。”

高竞即使不在俱乐部，也有目标和条件，不是战术菜单。

### 决策前互动：玩家选择要重放的失败

玩家选择一个片段，问教练组“如果这里失败，谁承担、下一次怎么处理”：

- 高位压迫被一脚打穿。
- 中低位先丢球后无法提速。
- 年轻球员第一次冒险传球直接造成反击。

人物的回答暴露他们是否真正接受方案的代价。玩家决定的不是阵型，而是允许哪种错误、给谁多长时间。

### 最终决定

#### 路线A：完整支持韩立锋的控制型足球

玩家原话：

> “比赛模型由你决定。输球以后我先问执行出了什么问题，不在记者面前替你换打法。”

**条件与代价**

- 防守距离和短期凝聚力更稳定。
- 进攻依赖定位球与少数创造者。
- 程野短期出场路径收窄。
- 若第二集没有签中卫，梁一川回撤会进一步降低进攻连接。

#### 路线B：确立主动压迫的俱乐部模型

玩家原话：

> “六周内允许身后出现代价。我们评估执行，不用一场比分撤回授权。”

**条件与代价**

- 前场夺回球权和年轻人路径增加。
- 体能消耗、身后空间和早期失误增加。
- 若第二集清理了两名替补，阵容深度不足会显著放大后半场风险。
- 韩立锋认为自己的方法被重新定义；是否转为反对取决于玩家此前是否尊重边界。

#### 路线C：确定三条共同原则

玩家原话：

> “阵型由教练定。落后时整体前压，年轻人进入真实轮换，赛后同时看结果和执行。”

**条件与代价**

- 能结合现有阵容调整，梁一川最容易在更衣室翻译。
- 权力边界更模糊，需要持续沟通。
- 一旦玩家以后临时加入第四、第五条原则，所有人都会认为共同原则只是另一种越级管理。

### 季前赛：第一场完整因果结算

比赛对手海港城采用边路快速推进。系统从前三集生成4—6个事件。以下为组合示例，不是固定剧本。

#### 例一：董事会目标 + 即战力中卫 + 控制型足球

- 新中卫两次处理传中，球队保持0比0。
- 前六压力让球员在最后十五分钟更急于向前，队形第一次拉开。
- 替补深度不足导致边后卫疲劳后无人可换。
- 随机：对手一次射门击中门柱。
- 可能比分：0比0或1比0；无论结果如何，报告都指出“新中卫帮助”和“阵容变薄”同时存在。

#### 例二：现金优先 + 未补中卫 + 主动压迫

- 前场两次抢回球权，程野获得一次射门。
- 现有中卫速度不足，被对手越过一次。
- 青训宿舍不会直接帮助本场，却让程野的训练恢复更稳定；这是长期因果，不强行变成本场加成。
- 随机：裁判对一次身体对抗未判犯规。
- 可能比分：1比1、2比1或1比2；玩家能理解比赛为何开放。

#### 例三：能力建设 + 三条共同原则

- 数据团队赛前指出边后卫体能风险，球队提前准备换人。
- 七十分钟落后后，中场按共同原则前压，制造一次机会。
- 进攻人才没有增加，机会不保证转化成进球。
- 随机：对方门将完成一次超常扑救。
- 可能比分：0比1或1比1；即使输球，玩家也能看到投入避免了哪一种更坏结果。

### 赛后页面

不再只显示比分。页面顺序为：

1. 比分和4—6个关键时刻。
2. “你的决定如何进入比赛”：列出两个帮助、一个代价。
3. “比赛中无法由你控制的事”：列出一个随机因素。
4. 两名立场不同人物的赛后解释。
5. 下一集会继续存在的身体、合同、现金和关系问题。

## 十一、文本与对白重写规范

### 11.1 场景先有动作，再有解释

避免：

> 这不是两种阵型，而是两种责任分配方式。

优先：

> 韩立锋把代表边后卫的磁块推回中线。高竞的语音还在播放：“再往前十米。”梁一川伸手按住了磁块：“先说丢球以后谁挨骂。”

主题由人物冲突生长出来，不由旁白替玩家总结。

### 11.2 每个人必须有不同的句法和词汇

- 周绍庭：期限、责任人、可出售的故事。
- 方雯：日期、金额、谁在当天等钱。
- 韩立锋：位置、动作、下一场谁上。
- 许青禾：票数、席位、程序和谁授权。
- 梁一川：球员是否还会传第二次、补第二次、先开口。
- 罗志衡：具体孩子、家庭、训练日程。
- 程野：短句、疑问、尚未形成漂亮观点。
- 孟书宁：风险、时间、不可逆身体后果。

### 11.3 删除“所有人都像作者”的信号

- 每场最多允许一句总结性旁白。
- 避免连续使用“不是……而是……”“不能证明……只说明……”。
- 不让每个人都能说出完整、平衡、带正反两面的政策分析。
- 人物可以说不完整的话、回避、停顿、误解问题或先保护自己。
- 重要事实可以通过文件、动作和后果出现，不必全部由对白解释。

### 11.4 选择文字必须可被玩家当场说出口

选择按钮第一行是玩家的原话，第二层才显示具体执行内容和风险。避免把一整段制度设计写成按钮标题。

## 十二、数据与程序改造需求（待本方案确认后实施）

### 12.1 人物数据

建议新增：

```json
{
  "agenda": {},
  "speech": {},
  "habits": {},
  "preferredOutcomes": [],
  "redLines": [],
  "behaviorState": "open",
  "memories": []
}
```

人物反应不再只有 `characters: { id: +12 }`，而要能写入：

```json
{
  "characterEffects": {
    "qiao": {
      "addMemories": ["ignored_cash_warning"],
      "setBehavior": "oppositional",
      "nextAction": "request_board_minutes"
    }
  }
}
```

### 12.2 财务数据

需要从四个总数扩展为：

- 现金账户与受限账户。
- 带日期的应付款、确定应收款和不确定应收款。
- 工资、分期、借款的付款计划。
- 现金预测和危机触发器。
- 每笔钱的来源、用途和关联人物。

### 12.3 因果账本

所有重要效果写入 `causalLedger`：

```json
{
  "sourceDecision": "e2_first_team_push",
  "domain": "squad_depth",
  "direction": "risk",
  "text": "为注册新中卫清理两名边缘球员，替补深度下降",
  "canAffect": ["late_match_fatigue", "injury_cover"],
  "expires": "season_end"
}
```

比赛事件、后续人物场景和结局都从这本账中引用来源，避免余波只是预写的装饰文本。

### 12.4 比赛报告

每个比赛事件保存：

- 时间。
- 发生了什么。
- 主要参与的竞技因素。
- 关联旧决定。
- 是否包含随机因素。
- 对比分或伤病的影响。

### 12.5 新手解释

增加术语表和 `seenTerms`，正文通过术语标记引用。解释层可关闭，但不能维护两套剧情文本。

## 十三、前三集样板的验收标准

找至少一名不熟悉足球的玩家和一名熟悉足球的玩家，完成前三集后分别检查：

### 人物

- 能否不看人物卡说出至少四个人分别想要什么。
- 能否指出两个人虽然都说真话，却为什么提出相反建议。
- 能否察觉某个人因为被绕过而改变了沟通方式。

### 足球理解

- 新手能否用自己的话解释更衣室、轮换、中卫、注册和高位压迫。
- 新手是否知道这些词为什么影响刚才的决定，而不只是背定义。

### 因果

- 玩家能否从季前赛中指出至少两个由自己旧决定造成的时刻。
- 玩家是否能够区分经营影响和不可控随机事件。
- 玩家是否仍觉得比分是“系统随机发的”。

### 金钱

- 玩家能否说出下一笔付款是什么、何时到期。
- 玩家选择支出前是否知道最坏情况下会缺多少钱。
- 现金紧张是否真实改变了可选方案或谈判条件。

### 文本

- 玩家能否仅凭对白分辨周绍庭、方雯、韩立锋和许青禾。
- 是否仍频繁出现“像在读政策说明”和“所有人说话一样”的反馈。
- 单次阅读段落是否足够短，人物动作和冲突是否先于概念解释。

只有在以上项目大部分通过后，才将相同系统扩展到第四至第十集。

## 十四、实施顺序

1. 确认本蓝图中的人物欲望、恐惧、偏好和红线。
2. 将前三集改写成可直接进入 `story-data` 的完整台词与分支稿。
3. 改造人物状态、现金时间表、因果账本和比赛事件数据结构。
4. 改造前三集界面与程序，只接入前三集样板。
5. 进行足球新手 / 熟悉足球双人试玩。
6. 根据试玩修改人物、术语密度、因果说明与随机幅度。
7. 扩展第四至第十集，并让既有人物状态和因果账本继续生效。

在第1步确认前，不开始批量改写台词或程序。
