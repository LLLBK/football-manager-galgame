# 《你受谁委任》视觉设定档案

这份文件只约束序章视觉素材。剧情正文、对话、选项和后果继续以 `story-data.json` 为唯一来源；视觉系统不得复制、修改或遮蔽剧情。

## 统一美术方向

- 当代中国职业足球题材的高品质日系动漫视觉小说插画：清晰线稿、赛璐璐与柔和厚涂结合、电影化光影。
- 保留成年人的真实年龄差、职业服装和体育职场剧气质；不采用校园萌系、幼态脸或夸张卡通比例。
- 岚城联是中等规模沿海城市俱乐部：专业但并不奢华。
- 画面主色为深绿、灰白和少量旧金色；避免真实俱乐部、品牌与可读水印。
- 背景与立绘分别生成。背景不包含主要人物，人物立绘不包含场景。

## 原文规定与允许补充

### 周绍庭

原文规定：俱乐部主席；地产商出身；五年前救下俱乐部；习惯把愿望写成指标；重视现金、排名、故事与可出售资产。

允许补充：约五十岁，中国男性；身形挺拔；深灰色西装；短黑发夹少量灰；不戴眼镜；表情克制，权威来自控制感而非夸张愤怒。

### 许青禾

原文规定：球迷信托理事；由8437名缴费会员一人一票选出；代表会员大会通过的三项底线；带着父亲留下的褪色东看台季票；不把共同记忆当成无主资产。

允许补充：约三十八岁，中国女性；实用的深绿夹克与旧围巾；不作商业精英造型；随身携带旧季票；目光直接但不咄咄逼人。

### 韩立锋

原文规定：一线队主教练；上赛季把球队从降级区带到第八；更衣室服他；把外部干预视为背叛；强调首发、更衣室与训练属于教练权限。

允许补充：约四十五岁，中国男性；运动员体格；深绿训练服；短发；长期在户外工作留下轻微风霜感；不穿西装。

### 方雯

原文规定：财务负责人；在俱乐部工作十一年；关注花钱以后哪一天会出事；会把承诺、到期日和付款写进表格。

允许补充：约四十岁，中国女性；简洁职业装；低调、精确；头发束起；不做冷酷反派化处理。

## 场景约束

### 董事会议室

上午九点；桌上有钢笔与两份文件；玻璃窗外训练场正在浇水；玻璃能反射演示文稿的光。会议室属于中等规模职业俱乐部，不能像跨国银行总部。背景不生成可读文字，原文中的目标由对话承担。

### 楼梯口

位于董事会议室门外；能感到训练基地的实用建筑质感；许青禾没有被安排坐进董事会议室。画面要保留“门内权力、门外代表”的空间差异。

### 训练场

七月下午；一线队训练正在进行；远处可以有模糊球员和训练标志物，但不能出现可辨识真实队徽或品牌。韩立锋处于工作状态。

### 新闻发布厅

上任发布会；规模专业但克制；有记者席、话筒与深绿俱乐部视觉元素。不得生成难以控制的可读赞助商文字。

### 财务办公室

发布会后；实用而长期使用的办公室；文件、表格与台灯比装饰更突出。不得表现成银行柜台或豪华董事办公室。

## 生成与复核规则

1. 使用 GPT Image 2 生成背景和角色母版。
2. 人物采用同一张三表情设定图切分，确保脸型、服装和比例一致。
3. 立绘先使用纯洋红背景生成，再在本地去背；正式素材保留透明通道。
4. 每张图必须先检查原文事实，再检查新增细节是否与人物身份冲突。
5. 视觉素材缺失时，网页必须退回原有纯文字显示，剧情不得中断。
