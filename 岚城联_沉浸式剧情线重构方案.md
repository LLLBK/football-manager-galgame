# 《岚城联：赛季前夜》沉浸式剧情线重构方案

## 一、先说结论

当前版本需要修改的不是“每道题再写长一点”，而是整个叙事单位。

现在的叙事单位是：

> 介绍一个经营主题 → 摆出两到四个方案 → 提前解释每个方案的收益与代价 → 立刻显示属性变化 → 进入下一题。

应该改成：

> 同一批人物在一个完整赛季里反复相遇 → 问题先以细节和关系出现 → 玩家主动询问信息 → 在信息不完整时作出承诺 → 后果通过比赛、人物行为和旧承诺延迟回来 → 赛季末才看清自己经营出了一家什么样的俱乐部。

十个阶段仍然可以保留，但每个阶段不再是一道题，而是一集短剧。十集共同讲述同一个赛季。

## 二、当前版本为什么仍然像问卷

### 1. 玩家在选择前已经看见“答案”

当前每个选项都会显示：

- “你会做什么”；
- “金额怎么来”；
- “代价与收益”；
- 现金、经营、竞技属性的精确增减；
- 会打开什么后续分支；
- 最终获得什么路线标签。

这会把一个本来可以很困难的决定，变成一道透明的资源配置题。真实管理者当然知道合同价格，但不会知道“球迷信任－7”“更衣室凝聚力＋4”，更不会提前知道半年后究竟由哪一个变量造成失败。

**修改原则：财务事实公开，人物与未来后果不公开。**

玩家可以知道账户只有多少钱、合同要付多少钱、哪天到期；但只能从人物的话、训练场细节、媒体风向和既往承诺推测信任、士气与风险。

### 2. 每个阶段只有一个场景、一次选择

当前一个阶段通常是两三段介绍，加一个决策按钮。玩家没有机会：

- 先观察异常；
- 分别听主教练、财务主管、球员和队医怎么说；
- 追问一两个问题；
- 发现不同人掌握的信息并不一致；
- 在决定前形成自己的判断。

所以玩家不是“进入事件”，而是“读完题干”。

**修改原则：每阶段至少包含四个节拍。**

1. 冷开场：先发生一件具体的小事，不解释意义。
2. 对质：两个利益不同的人给出冲突版本。
3. 调查：玩家选择向谁追问，得到不完整信息。
4. 决定与余波：采取行动，出现一个即时反应，真正后果留到以后。

### 3. 人物只有职位，没有生活史、立场和记忆

现在除了梁一川，主要说话者都是“主席”“财务主管”“主教练”“球迷代表”“队医”。他们更像不同指标的发言窗口：主席代表耐心，财务主管代表现金，球迷代表代表信任。

真实故事之所以动人，是因为 Guardiola、Mourinho、Messi、Bartomeu、Klopp、Keegan、球探 Walsh、St. Pauli 球迷都不是参数。他们各自有履历、尊严、误判、欲望和可失去的东西。

**修改原则：把抽象指标装进反复登场的人。**

同一人物必须在至少三个阶段出现，并记得玩家过去怎么对待自己。球员不会因为下一阶段开始就忘记工资承诺，主教练不会忘记玩家曾越过他签人，球迷也不会忘记玩家在发布会上说过“看台名字不会卖”。

### 4. 选择之后没有真正写过的剧情

当前54个选择没有一条专门创作的 `afterScene`。结果由通用模板根据属性变化拼接，因此不同选择会反复出现“会议纪要”“打印机吐出现金表”“论坛开始转发”“更衣室门关得更快”等相似反馈。

这些文字有气氛，却没有独特事件，也没有新的信息。

**修改原则：每个主决策至少写三层后果。**

- 当晚：一个具体人的即时反应；
- 两个阶段后：决定以意外方式回来；
- 赛季末：显示机会成本，告诉玩家当初没有选择的路后来意味着什么。

### 5. 十阶段只覆盖两个月，没有形成赛季故事

当前时间线从7月1日走到9月2日，最后只踢一场主场和一场客场。玩家还没看到冬季、伤病潮、一月窗口、冲刺和问责，游戏就宣布完成“赛季前10个阶段”。

这会造成两个问题：

- 选择来不及长出后果；
- 球队没有共同经历，人物关系不可能真正改变。

**修改原则：十阶段跨越整个赛季。** 第一场主客场只是第6阶段“开局六周”的一部分，不再各占一章。

### 6. 真实案例被直接解释，反而打断代入

目前场景会在正文中告诉玩家“这个母题来自巴萨、曼联、曼城、拉玛西亚……”这对教学有帮助，却会突然把玩家从岚城拉回课堂。

**修改原则：游玩时不讲出处，赛后复盘再揭示原型。**

游戏内只出现岚城联的人和事。完成章节或整局后，再由“真实足球档案”说明：这一压力结构与 Guardiola 选帅、St. Pauli 抗议或 Liverpool 重建有什么相似之处。

### 7. 玩家身份与权力边界不够清楚

玩家名义上是“足球事务经理”，却能直接决定首发轮换、战术投入、商务套餐、医疗方案和换帅。权力过于全能，冲突就很难真实。

**修改原则：玩家是足球事务总经理，而不是主教练。**

玩家能决定预算、合同、部门负责人、制度和重大危机处置；比赛名单属于主教练。玩家可以施压、建议、授权或撤换教练，但不能像在菜单里一样亲自更换五名首发。这样主教练才会成为真正的人，而不是战术按钮。

## 三、重构后的核心体验

玩家在一小时内应该产生五种感受：

1. **“我认识这些人。”** 梁一川敲门时，玩家知道他为什么不直接谈钱；财务主管沉默时，玩家知道账上可能有没说完的事。
2. **“我不确定谁是对的。”** 主帅、队医、球员和经纪人都有一部分真相，也都有自己的利益。
3. **“这是我说过的话。”** 过去的承诺会被人物原句引用，玩家无法把每一章当作新题重做。
4. **“这件事不是刚刚才发生。”** 冬季危机能追溯到夏窗、工资结构、主帅权限和季前负荷。
5. **“我赢了也可能做错，输了也未必做错。”** 比赛有偶然性；赛季末同时评价结果、过程和未来资产。

## 四、一条贯穿全季的主线

### 主线命题

岚城联不是一家濒临倒闭的俱乐部，也不是可以买遍市场的豪门。它最危险的地方是：每个部门都还能维持，每个人也都能给出一个听起来合理的方案，但俱乐部已经连续三年没有共同方向。

新赛季开始前，三件事同时到期：

- 队长梁一川只剩最后一年合同，争冠队江东竞技给出1600万正式报价；
- 现任主教练只剩最后一年合同，董事会内部已经有人接触名帅高竞；
- 主场东看台需要完成安全与设施改造，曜石集团愿意出钱，但要求冠名、压缩公益票并增加球员商业活动。

这三件事对应同一个问题：

> 岚城联准备拿什么换取眼前的竞争力，又准备保留什么，让它三年后仍然是岚城联？

### 赛季不是三条独立支线

梁一川、主教练和东看台必须互相缠绕：

- 留下梁一川会影响工资上限、主帅权限和新援角色；
- 换帅会改变梁一川是否愿意续约，也决定年轻前锋程野有没有位置；
- 拒绝大赞助会让一月窗口更紧，接受则可能引发球迷抗议和球员商业负担；
- 医疗危机发生时，玩家过去建立的主帅权威、队长关系和阵容厚度会同时决定可选项。

这样玩家经历的是一个因果网，而不是十个专题。

## 五、固定登场人物与人物弧

人物不宜过多。核心六人足以承载绝大部分指标。

### 1. 主席周绍庭

- **公开目标：** 两年内进入前四，让董事会和新资本相信俱乐部仍有上升空间。
- **私人压力：** “连续三年中游”也是他任期的失败；他不能再向董事会申请一笔没有结果的投入。
- **优点：** 愿意授权，理解俱乐部需要变化。
- **盲点：** 容易把清晰目标误认为可控结果，低估足球偶然性。
- **人物弧：** 从给玩家授权，到在冬季危机中想收回授权；最终是否继续信任玩家，取决于玩家有没有诚实管理预期。

### 2. 财务主管方雯

- **公开目标：** 让俱乐部按时支付工资和所有已签合同。
- **私人压力：** 上一届管理层曾让她用乐观收入预测掩盖现金紧张，她不愿再替任何人粉饰数字。
- **优点：** 准确、冷静，知道每一笔分期何时到期。
- **盲点：** 会把不能精确量化的价值压得过低。
- **人物弧：** 起初只提供数字；如果玩家愿意面对坏消息，她会逐步提供更深的风险判断。若玩家逼她为决策背书，她可能在董事会听证时保持沉默。

### 3. 主教练韩立锋

- **公开目标：** 证明过去三年的中游不是无能，而是在薄阵容下守住底线。
- **私人压力：** 合同只剩一年，已经听到俱乐部接触高竞的传闻。他担心新总经理用“现代化”之名架空自己。
- **优点：** 熟悉球员，能维持秩序，临场稳健。
- **盲点：** 信任老将，面对新方法容易防御，长期战术开发不足。
- **人物弧：** 可以成为愿意调整的长期合作者，也可以因玩家越权签人、公开施压而变成消极执行者；冬季危机是关系总清算。

### 4. 队长梁一川

- **身份：** 31岁，岚城青训出身，效力九年，是队内最有创造力的球员。
- **公开目标：** 知道俱乐部到底要争成绩还是重建，并在新闻发布前得到本人沟通。
- **私人压力：** 这是职业生涯最后一次加盟争冠队的机会；留下可能成为俱乐部传奇，也可能再浪费一年。
- **优点：** 比赛阅读和队内影响力仍然最高，愿意帮助年轻人。
- **盲点：** 已经习惯球队围绕自己运转，会把尊重与特殊地位混在一起。
- **人物弧：** 续约、约定再留一年、自由离队或夏窗出售都可以成立；关键是玩家是否守信，以及球队是否真的建立了他愿意相信的计划。

### 5. 青年队教练罗志衡与18岁前锋程野

- **罗志衡的目标：** 让梯队和一线队使用可衔接的训练语言，不再把青训只当公关口号。
- **程野的处境：** 有优秀的禁区跑位，却尚未适应成年对抗。他不是“潜力值”，会紧张、会犯错，也会记住谁第一次相信他。
- **人物弧：** 若得到真实机会，他们可能成为俱乐部新方向；若只在发布会中被用作重建象征，他们会让“青训承诺”变成反噬玩家的证据。

### 6. 球迷代表许青禾

- **身份：** 本地中学教师，父亲曾参与俱乐部早期志愿工作；她那条磨毛围巾不是造型，而是家庭记忆。
- **公开目标：** 保住东看台名称、青年票和定期球迷会议。
- **私人压力：** 她必须向不同球迷解释，为什么与管理层谈判不是被收买。
- **优点：** 理解社区，也理解球队需要收入。
- **盲点：** 有时会把妥协视为背叛，难以代表年轻的线上球迷和传统看台两种声音。
- **人物弧：** 可以成为危机中的沟通桥梁，也可以组织类似 St. Pauli 的持续抗议；取决于玩家是提前协商，还是只在需要气氛时才想起球迷。

### 7. 队医孟书宁（关键配角）

- **目标：** 让医疗意见不再只在球员倒下后被重视。
- **冲突：** 她掌握数据，却不掌握出场权；球员还可能相信私人理疗师，主帅则要为下一场结果负责。
- **人物弧：** 医疗信任取决于玩家是否保护专业判断，而不是只取决于投入多少钱买设备。

## 六、重构后的十集剧情

### 第1集：你受谁委任

**时间：7月1日。** 真实原型结构：巴萨会员政治、曼城新所有者授权、曼联所有权争议。

**冷开场。** 玩家到达体育场时，门外有人在重新描东看台旧壁画。工人说俱乐部还没通知他们新赛季是否保留。玩家此时不知道，看台冠名已经进入赞助谈判。

**主要场景。**

1. 周绍庭先单独见玩家，要求在发布会上承诺前四，交换800万追加预算。
2. 许青禾拿着上一届管理层签过但从未执行的球迷会议纪要，问这次承诺是否仍然只是公关。
3. 方雯提醒，800万是附带考核的董事会资金，不是俱乐部赚到的自由现金。
4. 玩家可以分别追问：前四失败会发生什么、球迷真正不可接受的底线是什么、俱乐部目前还有哪些未披露付款。

**主决定。** 不是选择“稳健/野心/社区”人格，而是确定书面授权：成绩承诺、预算边界、球迷参与权中，哪一项成为具有约束力的条款。

**即时余波。** 发布会上，记者引用玩家刚签的原句提问。回答方式决定外界记住的是“前四”“三年计划”还是“东看台不会被越过”。

**延迟回收。** 第5集赞助谈判、第8集董事会危机、第10集听证都会重新展示这份纪要。

### 第2集：账上有钱，未来没有

**时间：7月3—7日。** 真实原型结构：巴萨短债与递延工资、Derby球场交易、Liverpool债权人主导出售。

**冷开场。** 方雯把三张付款日历铺在桌上。4200万现金没有错，但其中一部分事实上已经属于工资、旧转会分期和东看台安全工程。办公室窗外，青训主管正在为坏掉的力量器械打电话。

**新增真实约束。** 为了支撑完整赛季，财务不能只有当前现金：

- 每月工资，而不是只支付一次“首月工资”；
- 前任管理层签下的转会分期；
- 梁一川续约或离队的奖金、工资与替代成本；
- 东看台许可工程的最低支出；
- 赞助款的到账时间和限制用途；
- 赛季目标奖金以及冬窗前的最低现金缓冲。

**主要冲突。** 周绍庭希望沿用乐观上座率预测，让转会预算显得充足；方雯要求按保守收入编制。青训和主帅都声称自己的项目不能再推迟。

**主决定。** 玩家决定向董事会提交哪一种预算，以及先冻结什么。关键不是“投一线队/青训/平均分”，而是是否愿意把坏消息提前写进预算，并由谁承担第一刀。

**即时余波。** 被冻结的部门不会只掉属性：对应人物会取消一次计划、停止招聘或把设备故障继续带入训练。

**延迟回收。** 第4集交易截止日会发现“账面余额”和“可动用现金”不同；第8集一月窗口会兑现递延付款。

### 第3集：球队准备怎样赢

**时间：7月8—18日。** 真实原型结构：巴萨在 Guardiola 与 Mourinho 间选帅、Liverpool 数据部门与 Klopp 的组织一致、曼城重建后台。

**冷开场。** 玩家旁听一堂训练。韩立锋让主力反复演练熟悉的低位反击；另一块场地上，罗志衡带青年队用不同方式压迫。梁一川在两块场地之间停了一下。

**主要场景。**

1. 韩立锋解释自己为何不愿贸然提速：老将体能和中卫转身速度不支持。
2. 罗志衡用一名梯队球员展示衔接方案，但承认自己没有顶级联赛临场经验。
3. 高竞团队递来完整改革计划，要求换六名工作人员、得到转会否决权并接受开局波动。
4. 玩家可让分析人员复盘韩立锋过去十场：结果普通，但机会质量可能没有排名那么差。

**主决定。** 决定的不只是“谁当教练”，而是足球决策结构：主帅、总经理、球探、数据和青训中，谁提名、谁否决、谁对失败负责。

**即时余波。** 被留下的人会根据权限变化作出反应。若保留韩立锋却公开接触高竞，他可能接受结果但不再分享真实判断；若换帅，老将会立刻询问自己是否还在计划中。

**延迟回收。** 第4集的引援名单必须由这一模型产生；第8集不能在危机中假装从未决定过权限。

### 第4集：截止日前的72小时

**时间：8月末。** 真实原型结构：Neymar离队后巴萨失去谈判主动、Suárez与Coutinho两次出售的不同结果、Manchester United 与 Ronaldo 的延期承诺。

**冷开场。** 晚上10点14分，江东竞技发来对梁一川1600万的正式报价，次日中午失效。十五分钟后，媒体记者打电话求证。俱乐部还没有告诉梁一川。

**主要场景。**

1. 经纪人表示梁一川没有提交转会申请，但希望知道球队计划。
2. 韩立锋说失去梁一川后现有体系没有第二个组织点；若留下，也不能在合同中承诺永远首发。
3. 方雯说明：卖出后并非1600万全部可用；经纪佣金、旧分成和替代者首期付款会吃掉一部分。
4. 球探给出三种不等价替代：32岁即战力韩拓、20岁罗钧、租借顾南。每人都有比赛、工资、适配和人物信息，而不是一行“老将/年轻/租借”。
5. 玩家只能选择两个人深入询问，不能一次获取全部隐藏信息。

**主决定。** 留下并续约、约定再留一年后按条件放行、立即出售并按竞技模型补位。任何方案都必须先决定由谁亲口告诉梁一川，以及对外怎样解释。

**即时余波。** 不能使用通用反馈。每条路线需要独有场景：签字时梁一川对“核心条款”的反应；延期承诺中的握手和沉默；转会后球迷商店门口仍挂着旧海报。

**延迟回收。** 第6集形成队内权力，第8集检验承诺，第10集计算机会成本。

### 第5集：东看台是谁的

**时间：9月上旬。** 真实原型结构：St. Pauli Social Romantics、巴萨 UNICEF 与 Qatar、曼城园区与社区再生。

**冷开场。** 工作人员试装曜石集团的新看台标牌，照片被球迷拍到。俱乐部尚未签约，社交媒体已经出现“把名字还给我们”。

**主要场景。**

1. 曜石给出足以解决安全工程并补充冬窗的报价，但要求冠名、减少公益票、增加核心球员活动。
2. 许青禾提出可接受商业合作，但要求保留看台名、设置赞助伦理条款和球迷监督席。
3. 商务总监警告，拆分给六家本地企业的方案金额较低、执行复杂，不能保证及时到账。
4. 梁一川或新核心提醒，新增商业活动会挤压恢复时间；这使赞助与第7集医疗线相连。

**主决定。** 不是“钱多/钱少/等等”，而是谈判哪些权益可以卖、哪些不能卖，以及谁拥有未来复核权。

**即时余波。** 球迷可能接受、观望或组织化抗议。抗议要有具体行为：红色围巾、空置十分钟、会员提案、社区游行，而不只是“球迷信任下降”。

**延迟回收。** 第6集主场开局气氛、第8集现金压力、第10集俱乐部身份评价。

### 第6集：开局六周

**时间：9—10月。** 真实原型结构：Guardiola清理老将后的糟糕开局、Leicester整合Kanté、Liverpool以主帅吸引并融入Mané。

**冷开场。** 连续六轮的剪报、比赛片段和训练细节快速推进。结果由此前阵容、战术、体能和偶然性共同生成，不让玩家在赛前直接选择“加主场气势”。

**根据路线生成不同核心事件。**

- 高薪老将到来：训练迟到或角色承诺与队长冲突；
- 罗钧到来：能力不错但连续错失机会，媒体称他“不够级别”；
- 程野被提拔：首次替补上场失误，随后在青年队比赛进球；
- 梁一川离队：球队缺少最后一传，球迷把每场平局都归咎于出售；
- 新帅到来：过程指标改善但前两轮不胜，董事会只看积分。

**主决定。** 玩家处理的是“组织如何解释开局”：坚持过程、改变角色、追加临时签约或公开施压。玩家不能直接替教练排首发，但可以决定是否维护既定权限。

**即时余波。** 由具体人物回应。若玩家曾承诺青训机会，却在程野一次失误后下放，罗志衡会引用原承诺；若玩家守住他，老将可能觉得竞争标准不公。

**延迟回收。** 开局形成的权力联盟进入医疗危机和冬季换帅争议。

### 第7集：身体属于谁

**时间：10—12月。** 真实原型结构：巴萨赛程伤病率、Dembélé缺席检查、Bayern与荷兰足协的Robben争议。

**冷开场。** 七天第三场比赛前，孟书宁发现一名核心的肌肉指标低于安全线。球员的私人理疗师认为可以上，韩立锋说只让他踢60分钟，球员本人不愿错过德比。

**主要场景。**

1. 队医解释风险区间，但明确承认不能预测一定受伤。
2. 主帅解释如果核心缺阵，替补方案会怎样改变比赛。
3. 球员说明自己担心一旦休战就失去位置或触发不了合同奖金。
4. 若玩家此前接受额外商业活动，恢复数据更差；若阵容厚度较好，休战才真正可行。

**主决定。** 决定出场权限与医疗规则：队医拥有红线否决、主帅在风险区间拍板，还是由球员签字承担选择。也可同时处理私人医疗信息共享。

**即时余波。** 比赛结果与是否受伤不能完全确定。即使冒险未受伤，医疗信任也可能下降；即使休战输球，决定也未必错误。

**延迟回收。** 第8集危机中的可用阵容和第9集冲刺体能由此产生。

### 第8集：冬天，所有旧账一起到期

**时间：12月—2月。** 真实原型结构：巴萨2比8后的重建困局、Newcastle越过Keegan签人、Manchester City落后八分仍保留Mancini。

**冷开场。** 一场大败后，训练基地外出现要求换帅的标语。董事会会议提前，旧转会分期同周到期，梁一川或其替代者又处在争议中心。

**主要场景。**

1. 周绍庭拿着第1集的目标纪要，要求解释为什么没有达到阶段目标。
2. 韩立锋拿出第3集确定的权限文件，指责管理层从未真正按模型给他阵容。
3. 方雯说明换帅赔偿会吃掉一月补强资金；若接受大赞助，则有钱但球迷压力更高。
4. 球探推荐一个“关系型交易”，经纪人暗示现在签下可以打开未来市场；没有足够现场观察，形成Keegan式权力冲突。

**主决定。** 三种可成立路线：

- 换帅并承认进入新周期，不承诺立刻反弹；
- 保留主帅，但调整助教、权限和六场复核点；
- 无条件支持现体系，把有限资金投向阵容或医疗。

**即时余波。** 任何路线都有人离开或失去权力。危机不能用一句“管理层耐心－8”结束。

**延迟回收。** 第9集决定球队以什么状态进入冲刺，第10集评价玩家是否在压力下背叛自己的制度。

### 第9集：最后五场

**时间：3—5月。** 真实原型结构：City对QPR、Leicester冲刺、Liverpool 97分无冠。

**动态目标。** 根据前八集，球队进入三种之一：争前四、争中游上半区、避免下滑。课堂仍可共享“最后五场”框架，但压力文案、对手和董事会目标不同。

**冷开场。** 联赛目标、杯赛半决赛和球员身体状态发生冲突。玩家过去保留或失去的轮换球员、青年球员和核心都重新进入名单讨论。

**主决定。** 不是直接选择某场“全攻/轮换”，而是与主帅确定五场资源优先级：保联赛、押杯赛、或维持轮换。主帅拥有具体首发权，玩家承担制度和目标责任。

**比赛呈现。** 至少选一场做成短小但具体的比赛叙事：中场伤退、替补是否可信、看台是否支持、最后一次换人由此前关系决定。随机性保留，但每个关键动作都能追溯到赛季选择。

**后果表达。** 若高质量过程遭遇坏结果，不能让系统宣判玩家错误；若靠偶然赢球，也不能自动掩盖阵容老化和未来付款。

### 第10集：你留下了什么

**时间：赛季结束后一周。** 真实原型结构：Liverpool以过程和资产评价重建、巴萨保留Messi一年的机会成本、Brighton长期基础设施投入。

**三场结算戏。**

1. **董事会听证。** 周绍庭和方雯分别拿出成绩报告与三年现金日历。玩家解释哪些目标完成、哪些判断仍然成立。
2. **人物回访。** 梁一川、韩立锋、罗志衡、程野、孟书宁和许青禾根据路线出现。不是所有人都夸或骂；他们只说明玩家的决定怎样改变了自己的位置。
3. **下一周期决议。** 玩家在即战力、偿债、训练基地、球场和合同清理之间确定下一季第一优先级。

**双重结算。** 游戏同时给出：

- 本季结果：排名、杯赛、关键比赛；
- 俱乐部状态：年龄曲线、核心合同、工资梯队、未来付款、青训通道、医疗信任、主帅权限和球迷合法性。

**结尾。** 玩家离开办公室时，东看台是否仍用旧名、程野是否在灯下加练、梁一川的海报是否还在，不由结尾文案随机挑选，而由整季事实生成。教学数据和真实案例对照在故事结束后另页出现。

## 七、每一集的固定叙事模板

为了控制45—60分钟总时长，不需要把游戏写成视觉小说长篇。每集约4—5分钟，使用以下结构：

| 节拍 | 时长 | 玩家做什么 | 叙事作用 |
|---|---:|---|---|
| 冷开场 | 20—30秒 | 阅读一段现场或消息 | 先感到异常，不先听解释 |
| 两方陈述 | 60—90秒 | 听两名人物各说一面 | 建立利益冲突 |
| 追问 | 30—60秒 | 从3个问题中选2个 | 让信息获取本身成为选择 |
| 主决定 | 30秒 | 选择行动和沟通方式 | 作出不可逆承诺 |
| 即时余波 | 30—45秒 | 看见一个独有后果 | 给决定重量，但不揭晓长期答案 |
| 回声 | 后续章节 | 被旧人物或旧文件再次提起 | 形成连续故事 |

序章压缩到3个场景，不再连续六次只点“继续”；结尾约5分钟。这样十集仍可控制在一节课内。

## 八、一集完整示范：重写“核心球星续约危机”

下面不是最终台词，只用于说明新旧结构的差别。

### 场景1：未接来电

**8月29日，22:14。训练基地二楼。**

办公室只剩走廊尽头一盏灯。你的手机上有三个未接来电：江东竞技体育总监、梁一川的经纪人、本地记者梁策。

方雯把打印出来的报价放在桌上。1600万，明天中午十二点失效。她用手指压着文件最后一页：

“别先看总价。二次转会分成、经纪费用和本月工资付完，真正能在截止日前动用的，不到一千二百万。”

门外有人停了一下，没有敲门。玻璃倒影里，你看见梁一川从走廊经过。

### 场景2：三个人的版本

**韩立锋：**

“卖了他，我不会说球队完了。但你得告诉我，谁来接最后一脚。韩拓能进球，不能组织；罗钧还没踢过这个级别；顾南只肯租一年。别把三个不同的问题叫成三个替代者。”

**梁一川的经纪人：**

“梁一川没有申请转会。江东能争冠军，这也是真的。他只要求一件事：别等官宣了才让他知道自己被卖了。”

**周绍庭发来的语音：**

“1600万对31岁的球员不是坏报价。可如果你决定留，明天发布会你自己解释，为什么我们拒绝一笔能重建阵容的钱。”

### 场景3：玩家只能问两个问题

**问方雯：“如果不卖，续约底线是什么？”**

她打开工资表：“固定工资提到队内第一档还能做。经纪人要的三年和主力保证一起给，明年另外四份续约会全部拿它当起点。”

**问韩立锋：“如果再留一年，你能保证他的角色吗？”**

“我能保证他是重要球员，不能保证每场都围着他踢。如果合同替我排首发，那你也不必留我。”

**问梁一川：“你想走吗？”**

他很久才回：“我想知道这里明年会不会还是同一套说法。江东给我的是一次争冠，你给我的不能只是一张三年合同。”

玩家只能看到自己问过的回答。未询问的信息不显示，也不会由旁白补齐。

### 场景4：主决定

选项不显示属性变化，不用“收益与代价”替玩家总结。

**A｜“报价到此为止。方雯，把不含主力保证的三年合同发过去。”**

玩家明确留下梁一川，但把主帅权限留在合同之外。经纪人可能继续抬价，其他主力会在后续谈判对照这份工资。

**B｜“我不会在今晚卖你。再留一年；明夏达到约定报价，我亲自放你走。”**

以书面最低报价、沟通时间和不公开施压为条件，换取一个赛季。它可能成为互信，也可能成为下一次危机的证据。

**C｜“接受江东报价。但在回复他们之前，我先去找梁一川。”**

出售不可撤回。玩家保住了沟通尊重，并获得重建资金；球迷和球队将在比赛中判断替代方案。

这里还可以把“怎么做”与“怎么说”拆开。相同交易若由记者先泄露、由主席宣布或由玩家亲口告知，会产生不同人物后果。

### 场景5：独有的即时余波

**A路线。** 梁一川看完合同，把写着“自动首发”的附页推回经纪人面前：“这一条删了。”他愿意签，但门外已有两名主力的经纪人在问方雯何时可以谈新约。

**B路线。** 两人在空训练场边握手。梁一川说：“明年如果你改口，我不会再进这间办公室。”系统只记录一条可见承诺：`达到约定报价时允许离队`，不显示信任加减。

**C路线。** 第二天清晨，梁一川离开基地。没有记者追上他。体育场商店门口，一个孩子抱着印有他号码的旧球衣，问店员新赛季还能不能印同一个名字。

### 后续回声

- **第6集：** A路线可能形成双核心工资比较；B路线让梁一川评价玩家是否兑现竞技计划；C路线让每次创造力不足都被媒体重新解释为卖人后果。
- **第8集：** 若B路线收到新报价或球队陷入危机，书面承诺重新出现；若A路线换帅，新帅是否接受核心地位成为冲突；C路线则检验替代者和资金用途。
- **第10集：** 不只显示本季表现，还显示放弃的报价、支付的工资、替代者贡献、其他续约连锁反应和俱乐部信用。

这一集使用了 Ronaldo 延期离队、Barcelona 强留 Messi、Liverpool 两次出售核心的真实压力结构，但游戏内不需要提到这些名字。

## 九、玩家界面必须同步修改

只改文案、不改呈现方式，问卷感仍会回来。

### 游玩层：只展示角色真正知道的东西

**保留：**

- 当前现金、到期日、合同金额、工资承诺等确定财务事实；
- 已经公开的排名、赛程、伤病诊断；
- 玩家亲自获得的报告和对话；
- 自己签过的承诺与会议纪要。

**从选择按钮中移除：**

- “代价与收益”总结；
- 属性 `+8/-5`；
- “后续分支”标签；
- “稳健重建型”“商业冲刺型”等路线判词；
- 系统替玩家做出的价值判断。

**替换为：**

- 人物原话；
- 合同条款摘要；
- 可确认的支出和到期日；
- “你还不知道”的明确空白；
- 玩家已作承诺列表。

### 复盘层：教学信息在故事之后出现

一集结束后只显示简短“经理日志”：

- 今天决定了什么；
- 向谁作了什么承诺；
- 哪些付款已经锁定；
- 哪些问题仍未解决。

整局结束后再显示：

- 经营与竞技指标；
- 真实案例原型；
- 选择之间的因果链；
- 与其他同学球队的比较数据。

这样既保留教学价值，也不破坏游玩时的沉浸。

## 十、后台状态也要从“分数”升级为“事实”

现有数值可以保留用于计算，但剧情分支应优先读取以下事实型状态：

### 1. 承诺

```text
promises:
  - protected_east_stand_name
  - promised_top_four
  - lin_xiao_release_next_summer
  - youth_minutes_target
  - coach_controls_lineup
```

承诺有“已兑现、尚未到期、已违背”三种状态。人物会引用具体承诺，而不是只读取俱乐部文化分。

### 2. 人物关系

```text
characters:
  lin_xiao:
    trust: hidden
    grievance: "management_leaked_offer"
    leverage: "one_year_contract"
    knows: [coach_contacted_gu_wei]
```

玩家不需要看到数值，只通过人物是否主动来谈、是否通过经纪人、是否在公开场合支持管理层来感知。

### 3. 未解决问题

```text
open_threads:
  - january_installment_due
  - east_stand_safety_license
  - replacement_creator_missing
  - private_physio_conflict
```

它们会跨阶段继续存在，直到被解决、恶化或转化。

### 4. 信息与误判

记录玩家是否见过某份报告、问过哪个问题。玩家没问，不代表事实不存在；这会让尽调和沟通本身具有意义。

### 5. 计划回声

每个主决定都附带回调条件：

```text
callbacks:
  immediate: scene_after_sale
  stage_6: fans_blaming_missing_creator
  stage_8: january_offer_returns
  stage_10: opportunity_cost_report
```

## 十一、哪些现有内容值得保留

当前版本不是全部推倒重来，以下基础很好：

- 岚城联、东看台、旧海报、磨毛围巾等视觉意象；
- 梁一川“别让我从新闻里知道方向”这一核心诉求；
- 4200万现金和未来付款作为硬约束的思路；
- 韩拓、罗钧、顾南、程野、罗志衡这些可复用的人名与位置；
- 条件事件池的技术结构；
- 比赛有随机性、经营状态传导到竞技状态的原则；
- 最后导出球队数据供课堂联赛使用的功能。

需要重写的是：阶段时间线、人物档案、选择呈现、后果场景、跨阶段承诺和结局。

## 十二、建议的实际改写顺序

不要立刻一次重写全部十集。最稳妥的顺序是：

1. **先做第4集“截止日前72小时”的完整可玩样板。** 它同时检验人物、财务、询问、承诺、独有余波和延迟回调。
2. 暂时隐藏选项影响标签和精确属性变化，验证没有提示时玩家能否理解局势。
3. 为六名核心人物建立统一档案和说话方式。
4. 重写第1—3集，让玩家进入第4集前已经认识这些人并签下真实承诺。
5. 写第5—8集的回调，确保夏天的决定在冬天重新出现。
6. 最后做第9集比赛叙事与第10集双重结算。

第一轮验收不看文本有多长，只看三个问题：

- 玩家能否不用属性提示，说清楚每个人为什么这样要求？
- 玩家做决定时是否真的犹豫，而不是一眼看出最优解？
- 两个阶段后，玩家是否记得“这件事是我当时亲口答应的”？

如果三项成立，代入感和沉浸感才真正开始出现。

## 十三、与现有程序的衔接判断

这套方案不要求推翻现有计算系统，但需要更换剧情播放器。

**可以直接保留：**

- `eventPool + trigger + priority` 的条件事件选择；
- 财务、经营、竞技三组后台状态；
- `requires` 对现金和事实条件的硬限制；
- 比赛权重与随机波动；
- 结局导出和课堂联赛数据。

**必须调整：**

- 当前 `一阶段 = 一个事件 = 一次选择` 改为 `一阶段 = 一个 episode = 多个 scene + 一次主决定`；
- 增加 `question` 场景，让玩家只能选部分追问；
- 增加 `promise`、`character`、`openThread`、`knowledge` 和 `callback`；
- 每个主选择写独立即时余波，不再依赖通用反馈模板；
- 玩家界面不再调用影响列表显示精确属性增减；
- 将正文中的真实案例说明移到结局后的复盘页；
- 时间轴改成完整赛季，比赛模拟从两场扩展为开局段、冬季段和冲刺段的摘要与关键战。

因此，最适合的下一步不是先把30个真实案例全部写进程序，而是用第4集做一个新的“纵向切片”：只做一集，但把人物、追问、承诺、独有后果和后续回调接口全部做对。它通过试玩后，再按同一结构扩写其余九集。
